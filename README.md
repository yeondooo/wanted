### 🚀 프로젝트 실행방법
1. root폴더에 .env 파일을 생성합니다.
```
REACT_APP_API_BASE_URL=https://pre-onboarding-selection-task.shop
```
2. 의존성 패키지를 설치합니다.
```
npm install
```
3. 프로젝트를 실행합니다.
```
npm start
```
